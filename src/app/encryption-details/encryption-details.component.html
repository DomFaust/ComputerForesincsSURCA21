Welcome to the world of Encryption!

Below, you will find some of the basic terms that are used when discussing this field.

Plaintext
Ordinary, intelligible text, that is not formatted or written in code Cipher
An algorithm to encode and disguise plaintext, or to reveal encoded ciphertext Ciphertext
Text that has been encrypted with the use of a cipher Encryption
The process of securing digital data by using mathematical algorithms to encode the information Keys
In encryption, it is a chunk of information, usually a random string of numbers or letters, that can be used to encode or decode data Hash
A value derived from a hashing encryption process that cannot be used to reverse engineer the original plaintext MD5 (Message-Digest Algorithm)
A hashing algorithm that accepts a message as an input and returns a value as its output. That value can authenticate the original message. Purposes of the MD5
MD5 has been used to verify that a file has been unaltered. It has also ben used to securely store passwords.


Welcome to the JPEG section!

This section focuses on identifying and extracting information from these files.

Metadata
Information about a data structure, such as a file. It can define many things, including: modified time, created time, file ownership, and file size. JPEG
Also referred to as JPG. A file format that is used to compress image files. Exchangeable Image File Format (EXIF)
It is included in JPEG files and contains an abundance of information about the file itself. Python Image Library (PIL)
A library in the Python programming language that is built upon the Dictionary data structure. It is capable of extracting information from JPEG files and manipulating that information as a user chooses. Tags
An object that provides keys that are used to access dictionary elements. In the Python Image Library, there are multiple sets of Tags including EXIFTAGS and GPSTAGS. EXIFTAGS
This object provides information detailing the exact qualities and attributes of the image. GPSTAGS
This is the object that provides access to the dictionary items that identify the location-based information about an image. Requirements for GPS/EXIF Extraction
